## Beecrowd

Este repositório contém códigos desenvolvidos para a resolução de questões do BeeCrowd, focando no estudo de Programação Imperativa e Funcional, assim como Algoritmos e Estrutura de Dados.

## Estrutura do Repositório

O repositório está organizado em diretórios que representam diferentes categorias de questões no BeeCrowd. Cada diretório contém soluções para problemas específicos. A estrutura é a seguinte:

Imperative: Contém soluções que seguem o paradigma de programação imperativa.
Functional: Apresenta soluções que exploram o paradigma de programação funcional.
DataStructures: Engloba códigos relacionados a algoritmos e estruturas de dados.

## Sobre o BeeCrowd

O BeeCrowd é uma plataforma online que oferece uma variedade de desafios de programação, abrangendo diferentes níveis de dificuldade. Este repositório foi criado para armazenar as soluções desenvolvidas em linguagem C para os problemas encontrados no BeeCrowd.

## Como Contribuir

Se você deseja contribuir para este repositório, sinta-se à vontade para enviar pull requests. Certifique-se de seguir as melhores práticas de codificação, adotando um estilo claro e documentando o código quando necessário.

Ao contribuir, por favor, siga estas diretrizes:

- Crie uma branch específica para a sua contribuição.
- Documente suas alterações de forma clara e concisa.
- Certifique-se de testar suas alterações antes de enviar um pull request.
  
## Como Utilizar

Para utilizar este repositório, basta cloná-lo em sua máquina local e explorar os códigos disponíveis. Cada pasta possui seu próprio conjunto de problemas e soluções, organizados de forma a facilitar a navegação e entendimento.

```bash
git clone https://github.com/seu-username/BeeCrowd.git
cd BeeCrowd
```
